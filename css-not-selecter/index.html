CSS3選擇器 :not()

css selector
網頁設計
發表日期:   2020-01-29
總文長:   329

用途

假設有多個一樣 label 的元素要套用一種樣式，但又有幾個需要排除在外時使用。

例子

  • HTML:

    <div class="ex-box">
  <span class="ex">1</span>
  <span class="ex">2</span>
  <span class="ex">3</span>
  <span class="ex">4</span>
</div>

  • CSS

    .ex-box {
  background-color: #455A64;
  text-align: center;
}
.ex {
  color: #f00;
  font-style: italic;
  font-weight: bold;
}
1
2
3
4

可以看到上面：1~4 都有斜體跟粗體，為了方便觀察，這裡讓文字變為紅色

  • HTML:
    <div class="not-ex-box">
  <span class="not-ex">1</span>
  <span class="not-ex">2</span>
  <span class="not-ex" id="not-ex-ignored">3</span>
  <span class="not-ex">4</span>
</div>

接著將 3 號加上 not-ex-ignored 的 id。

  • CSS
    .not-ex-box {
  background-color: #455A64;
  text-align: center;
}
.not-ex:not(#not-ex-ignored) {
  color: #f00;
  font-style: italic;
  font-weight: bold;
}

並將此 id 略過（使用 :not 選擇器）

如此一來，就只有三號沒有套用到屬性。
