Perl 模組安裝

note perl
Note
發表日期:   2019-12-27
總文長:   121

使用自動安裝的環境

perl -MCPAN -e shell
  • 類似於 apt、brew 的好用套件，輸入指令會進入此環境中，成功的話終端機會顯示此畫面:
cman>

安裝

cman> install Module::Name
  • 直接下 install 指令 後面接模組名稱即可。
